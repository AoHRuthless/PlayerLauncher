PlayerLauncher

This GitHub page is dedicated to PlayerLauncher, a devBukkit Plugin. PlayerLauncher allows players to soar into the air with a command in Minecraft. PlayerLauncher [1.3.1] is compatible with Minecraft 1.2.5 and above, including 1.6.4. This means modpacks such as Tekkit Classic and Feed the Beast are compatible.

PlayerLauncher is useful for several functions such as:

  • A Donor perk
  • Raiding, PvP, etc.
  • Routine, dangerous tasks, such as fighting mobs or mining.
  • And more!

The page has been designed to offer more comprehensive support towards using this bukkit plugin, found at http://dev.bukkit.org/bukkit-plugins/playerlauncher/

Visit the Wiki for any help with the plugin: https://github.com/AoHRuthless/PlayerLauncher/wiki

I hope you enjoy using my plugins; and if you have any issues, post them at https://github.com/AoHRuthless/PlayerLauncher/issues
